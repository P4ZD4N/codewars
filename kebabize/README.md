# DESCRIPTION

<br><br>

<br>
"camelsHave3Humps" --> "camels-have-humps"
<br>
"CAMEL" --> "c-a-m-e-l"
<br><br>
Notes:
<br><br>
the returned string should only contain lowercase letters
